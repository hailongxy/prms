jQuery File Upload 文件上传插件支持多文件选择、文件拖拽、上传进度显示、上传验证、图片音频视频预览
以及跨域名、分块、客户端图像缩放以及可恢复的文件上传插件， 同时支持多个后台服务语言平台
(PHP, Python, Ruby on Rails, Java, Node.js, Go etc.) 也支持标准的Html Form表单上传

Demo Notes

  • 演示DEMO最大上传文件大小为1 MB (默认是没有限制).
  • 只能上传图片格式文件(JPG, GIF, PNG).
  • 上传文件自动删除在3分钟之后.
  • 你可以拖拽 文件从你的桌面到这个页面(看下浏览器支持).
  • 更多信息查看 项目网站 和 文档 .
  • 样式是基于Twitter's Bootstrap CSS 框架以及图标是来自Glyphicons.
